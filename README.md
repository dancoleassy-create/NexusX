## Описание
NexusX - многофункциональное меню для Roblox с поддержкой ESP, Aimbot и других фич.

## Функции
- RENDER: Fullbright, FPS, Optimization, MiniMap
- ESP: Отображение игроков и NPC через стены
- MOVEMENT: Fly, Noclip, Speed, Hitbox
- COMBAT: Aimbot
- TELEPORT: Телепорт по метке, телепорт к игрокам
- SETTING: Rejoin, Anti-AFK
